<h3 id="instalasi">Instalasi</h3>

1. Git Clone repository (master) berikut
```
https://github.com/Thahirudin/kl-frontend.git
```
2. Install NPM Packages
```
https://github.com/Thahirudin/kl-frontend.git
```
3. Jalankan Server!
<br>
Development
```
npm run start-dev
```
Production
```
npm run build
npm run serve
```
